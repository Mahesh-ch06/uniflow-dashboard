# EmailJS HTML Templates

Use these files as the HTML body in your EmailJS templates.

## Template Mapping

- `VITE_EMAILJS_TEMPLATE_FACULTY_WELCOME=template_faculty_welcome`
  - Paste HTML from `faculty-welcome-template.html`
- `VITE_EMAILJS_TEMPLATE_STUDENT_ABSENT=template_student_absent`
  - Paste HTML from `student-absent-template.html`

## Recommended EmailJS Subjects

### `template_faculty_welcome`
Welcome to UniFlow Dashboard - Faculty Account Credentials

### `template_student_absent`
Attendance Alert: Marked Absent in {{course_name}} on {{attendance_date}}
